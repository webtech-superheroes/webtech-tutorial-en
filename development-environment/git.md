# Working with GIT

Join GitHub \([https://github.com/join](https://github.com/join)\).

![working with git](../.gitbook/assets/001-working-with-git.jpg)

```bash
git init
git clone [url]
git status
git add [file]
git commit -m "a descriptive commit message"
git push [alias] [branch]
git pull
```
